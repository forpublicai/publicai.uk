# publicai.uk

Demo site for the UK AI Service proposal—a sovereign AI pilot for the public good.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

If build fails with Turbopack/SWC errors on Apple Silicon, try `next build --webpack`.

## Tech

- Next.js 16 (App Router)
- Tailwind CSS 4
- TypeScript
